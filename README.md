# 실행 방법
1. git을 통해 압축 파일을 내려 받는다

2. Visual Studio를 준비한다

3. 내려 받은 파일의 압축을 풀고 `ArrowStorm.vcxproj`를 더블클릭해 프로젝트를 연다

4. 프로젝트를 실행하면 게임이 실행된다

# 조작 방법
- WASD를 통해 플레이어 이동
- IJKL를 통해 플레이어가 바라보는 방향을 조절
- U로 화살 발사
- O로 스킬 사용
- ESC로 저장 및 게임 종료
